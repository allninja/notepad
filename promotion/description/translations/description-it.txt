# Application:
OI Blocco Note

# Promo text:
Crea e condividi le tue note con OI Blocco Note.

# Description:
OI Notepad è un'applicazione con fonte pubblica, con tutte le caratteristiche per scrivere le note. Con questa potete etichettare, cercare, e mandare le note, oltre alla funzionalità di base, come sarebbe creare, cancellare e modificare delle note. Le caratteristiche avanzate come sarebbe criptare, importare/esportare, e le memo vocali sono tutte disponibili attraverso le estensioni. Tutte queste funzioni sono portate insieme in un aspetto semplice e efficiente, personalizzabile con una varietà di temi. 

Questa applicazione non contiene la pubblicità e non richiede la permissione di internet. 

Il codice sorgente dell' applicazione gratuita e con la sorgente aperta è disponibile a: 
http://code.google.com/p/openintents

Per la lista completa con le modifiche e una lista con le domande frequenti, prego visitare: 
http://www.openintents.org

Potete aiutare a migliorare la traduzione nella vostra lingua a Launchpad: 
https://translations.launchpad.net/openintents/trunk

Caratteristiche: 
* Aggiungi, modifica e cancella le note.
* Premere prolungatamente sulle note per aggiungere le etichette.
* Cambia la dimensione del font e l'odine dell'elenco.
* Una struttura organizzata in modo chiaro con una varietà di temi.
* Cercare rapidamente attraverso le note.
* Mandare note come MMS oppure SMS.

Estensioni disponibili per "OI Notepad": 
* Registrate e allegate le memo delle voci alle note con "OI Voice Notes".
* Con "OI Convert CSV" importate/esportate le note dalla scheda SD.
* Mostra le informazioni della licenza con "OI About".
* Inserisci la data e l'ora alle note con "OI Insert Date".
* Capovolgere il testo con "Upside Down".

Applicazioni che funzionano con "OI Notepad": 
* Back up le note con "My Backup Pro".
* Criptate le note usando "OI Safe".
* Gestisci le note della vostra scheda SD con " File Manager OI".
* Aggiornate con "OI Update".
* Distribuisci le note attraverso "Facebook", "Gmail", "Google+", "Twitter", e tante altre applicazioni.
