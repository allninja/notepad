# Application:
OI Notepad

# Promo text:
Create and share your notes with OI Notepad.

# Description:
OI Notepad это полнофункциональный  менеджер заметок с открытым исходным кодом. С его помощью можно помечать, искать, а также отправлять заметки, в дополнение к базовым функциям, таким как создание, удаление и редактирование заметок. Дополнительные функции, такие как шифрование, импорт / экспорт, а также голосовые заметки все это доступно с помощью расширений. Все эти функции объединены в упрощенный и эффективный интерфейс, настраиваемый с помощью различных тем. 

Это приложение является бесплатным и не требует постоянного подключения к Интернет. 

Исходный код этой программы является свободным и открытым. Доступен по адресу: 
http://code.google.com/p/openintents

С полным списком изменений и списком часто задаваемых вопросов можно ознакомиться на: 
http://www.openintents.org

Вы можете помочь улучшить перевод прилоежния на ваш язык, посетите Launchpad: 
https://translations.launchpad.net/openintents/trunk

Возможности: 
* Добавление, изменение и удаление заметок.
* Долгое нажатия на записи чтобы добавить метку.
* Изменить размер шрифта и порядок сортировки.
* A clearly arranged layout with a variety of themes.
* Быстрый поиск записок.
* Отправка заметок как MMS или SMS.

Доступные расширения для "OI Notepad": 
* Запись и прикрепление голосовых сообщений к заметке с помощью "OI Voice Notes".
* Импорт/экспорт заметок из SD-карты с помощью "OI Convert CSV".
* Показать информацию о лицензии с помощью "OI About".
* Добавить дату и время в заметки с помощью "OI Insert Date".
* Перевернуть текст с помощью "Upside Down".

Приложения, которые работают с "OI Notepad": 
* Резервное копирование заметок с помощью "My Backup Pro".
* Шифрование заметки с помощью "OI Safe".
* Управление заметками на вашей SD-карте с помощью "OI File Manager".
* Обновить с помощью "OI Update".
* Поделиться заметками через "Facebook", "Gmail", "Google +", "Twitter", и многие другие приложения.
