# Application:
OI Notepad

# Promo text:
Creează și partajează noțitele cu Carnețel OI.

# Description:
OI Notepad este o aplicaţie cu sursa publică, cu toate caracteristicile pentru a lua notiţe. Cu ea puteți eticheta, căuta, şi trimite notele, pe lângă funcţionalitatea de bază, cum ar fi crearea, ştergerea si editarea notelor. Caracteristici avansate cum ar fi criptarea, importarea/exportarea, şi memourile vocale sunt toate disponibile prin extensii. Toate aceaste funcţionalităţi sunt aduse împreună într-un aspect simplist şi eficient, personalizabil printr-o varietate de teme. 

Această aplicație nu conține reclame și nu necesită permisiunea de internet. 

Codul sursă al acestei aplicații gratuite și cu sursă deschisă este disponibil la: 
http://code.google.com/p/openintents

Pentru lista completă cu modificări și o listă cu întrebări frecvente, vizitați: 
http://www.openintents.org

Puteți ajuta la îmbunătățirea traducerii în limba dumneavoastră la Launchpad: 
https://translations.launchpad.net/openintents/trunk

Caracteristici: 
* Adăugaţi, editaţi şi ştergeţi note.
* Apăsaţi lung pe note pentru a adăuga etichete.
* Schimbă dimensiunea fontului şi ordinea de sortare.
* Un aspect aranjat în mod clar cu o varietate de teme.
* Căutare rapidă prin note.
* Trimiteţi note prin MMS sau SMS.

Extensii disponibile pentru „OI Notepad”: 
* Înregistraţi şi ataşaţi memouri vocale la note cu „OI Voice Notes”.
* Cu „OI Convert CSV” puteţi importa/exporta note de pe cardul SD.
* Arată informaţiile de licenţă cu „OI About”.
* Inseraţi data şi ora în note cu „OI Insert Date”.
* Întoarce textul cu susul în jos cu „Upside Down”.

Aplicaţii care funcţionează cu „OI Notepad”: 
* Crează o copie de rezervă a notelor cu „My Backup Pro”.
* Criptează notele cu „OI Safe”.
* Administrează notele de pe cardul dumneavoastră SD cu „File Manager OI”.
* Actualizaţi cu „OI Update”.
* Distribuie note prin intermediul "Facebook", "Gmail", "Google+", "Twitter", și multe alte aplicaţii.
