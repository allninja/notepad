# Application:
OI メモ帳

# Promo text:
OI メモ帳 でノートを作成したり共有したりすることができます。

# Description:
OI メモ帳は完全なオープンソースメモ取りアプリケーションです。 これを使ってノートの作成・削除・編集などの基本機能のほかにも、ノートをタグ・検索・送信することができます。 暗号化、インポート・エクスポートやボイスメモなどは拡張機能を通じて利用することができます。 すべての機能はシンプルで効率的なレイアウトで表示され、様々なテーマでカスタマイズ可能です。 

このアプリケーションは広告を含まず、インターネット許可を必要としません。 

この無料でオープンソースのアプリケーションのソースコードは以下の場所から取得できます： 
http://code.google.com/p/openintents

すべての変更リストや質問リストを見るには： 
http://www.openintents.org

あなたもこの言語の翻訳を Launchpad で改善することができます： 
https://translations.launchpad.net/openintents/trunk

機能： 
* ノートの追加・編集・削除。
* ノートに長押ししてタグを追加。
* フォントサイズと並べ替え順序を変更。
* 様々なテーマを持つきれいにアレンジされたレイアウト。
* ノートを素早く検索。
* ノートを MMS か SMS として送信する。

"OI メモ帳" で利用可能な拡張子： 
* "OI ボイスノート" でノートに録音したボイスメモを添付。
* "OI CSV 変換" で SD カードからノートをインポート・エクスポート。
* "OI 情報" のライセンス情報を表示する。
* "OI 日付挿入" でノートに日付と時間を挿入。
* "逆さま" でテキストを逆さまにする。

"OI メモ帳"  と動作するアプリケーション： 
* "マイバックアップ プロ" でノートをバックアップ。
* "OI 金庫" を使ってノートを暗号化。
* "OI ファイルマネージャー" でSDカードのノートを管理。
* "OI アップデート" でアップデート。
* "Facebook", "Gmail", "Google+", "Twitter" などのアプリケーションを通してノートを共有。
